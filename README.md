## 🌟 Visão Geral

O **Montador de RPG** é o front-end avançado de uma plataforma idealizada para revolucionar a forma como a comunidade de RPG de mesa interage com mundos virtuais.

Mais do que um simples site ou painel, é uma ferramenta visual que permite:

- Montagem de campanhas complexas  
- Gerenciamento de sessões  
- Imersão interativa dos jogadores  

O projeto se destaca por integrar:

- Interfaces modernas com **React**
- Renderização **3D em tempo real com Three.js**

Tudo isso rodando diretamente no navegador 🚀
